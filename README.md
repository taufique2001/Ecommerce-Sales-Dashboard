# 📊 Interactive Sales Analytics Dashboard — Power BI

An interactive Business Intelligence dashboard built using **Power BI** to analyze sales performance, identify trends, and provide actionable insights for data-driven decision-making.

The project demonstrates an end-to-end analytics workflow including **data cleaning, ETL processing, data modeling, DAX calculations, and dashboard development**.

---

## 🚀 Project Overview

The **Interactive Sales Analytics Dashboard** analyzes business sales data across multiple regions and product categories. It helps stakeholders monitor key performance indicators (KPIs), understand sales trends, and evaluate business performance through interactive visualizations.

The dashboard transforms raw sales data into meaningful insights using Power BI capabilities.

---

## 🎯 Objectives

- Analyze overall sales performance and revenue trends
- Track business KPIs dynamically
- Identify high-performing regions and product categories
- Reduce manual reporting efforts through automated dashboards
- Enable data-driven business decisions

---

| Technology | Usage |

## 📂 Dataset

The dataset contains sales transaction records including:

- Order details
- Sales revenue
- Product categories
- Regional information
- Customer-related data
- Date-based sales records

## 🔄 Data Preparation & ETL Process

Performed data transformation using **Power Query**:

✔ Removed duplicate records  
✔ Cleaned missing/inconsistent values  
✔ Standardized category labels  
✔ Normalized date formats  
✔ Prepared structured data model for analysis  

Improved overall data quality and consistency for accurate reporting.
